# 🎵 2025 OKÜ Müzik Festivali Web Sitesi

Modern ve responsive tasarıma sahip, **Osmaniye Korkut Ata Üniversitesi** temalı festival tanıtım web sitesi.

## 📸 Proje Hakkında

Bu proje, ziyaretçilere festival hakkında bilgi vermek amacıyla geliştirilmiş tek sayfalık (Landing Page) bir web sitesidir.

Site içerisinde;

* 🏠 Ana Sayfa
* ℹ️ Hakkında
* 🎤 Sanatçılar
* 📅 Festival Takvimi
* 📞 İletişim

bölümleri bulunmaktadır.

## 🚀 Kullanılan Teknolojiler

* HTML5
* CSS3
* JavaScript
* Responsive Web Design

## ✨ Özellikler

* Modern kullanıcı arayüzü
* Responsive (Mobil Uyumlu) tasarım
* Akıcı kaydırma animasyonları
* Şık festival teması
* Kolay kullanılabilir menü yapısı

## 📷 Proje Görünümü

> Ana Sayfa

```
2025 OKÜ Müzik Festivali'ne Hoş Geldiniz

Osmaniye Korkut Ata Sunar

2025 Festivalleri Seni Bekliyor.
```

## 💻 Kurulum

Projeyi bilgisayarınıza klonlayın:

```bash
git clone https://github.com/MelihSezginn/music-festival-website-oku.git
```
veya direk dosyayı indirin

Ardından `index.html` dosyasını tarayıcıda açmanız yeterlidir.
